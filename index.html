Scope

To satisfy the ever-increasing demand for Cloud Computing resources from emerging applications such as Internet-of-Things (IoT), academics and industry experts are now advocating for going from large-centralized cloud computing infrastructures computing nodes located at the edge of the network. Mobile edge computing paradigm is expected to improve the agility of cloud service deployments in addition to bringing computing resources closer to end-users. In particular, fog computing with the features (e.g., low latency, location awareness, and capacity of processing large number of nodes with wireless access) to support heterogeneity and real-time applications is an attractive solution to delay- and resource-constrained large-scale wireless applications.

However, many fundamental questions arise at the same time. For example, how and where to process the learning data are significant issues due to the resource constraints for processing of the raw data at the master and controlled domain . Therefore, edge learning that is being actively investigated by many researchers in several technologies is yet to be applied to intelligent edge computing support engines. Moreover, the selection of a learning algorithm will become a significant factor to be considered. This half-day event offers a unique platform for industry leaders, innovators, and researchers from industry and academia to collaborate and exchange ideas that will help to drive rapid deployment in intelligent edge computing. The program committee solicits previously unpublished papers offering novel research contributions in any aspect of intelligent edge computing.

Topics of interest

This workshop seeks to attract high-quality contributions covering both theory and practice overall aforementioned and other related aspects of intelligent edge computing. The topics related to intelligent edge computing, but are not limited to:

  • >> Edge, fog, and mobile edge computing for intelligent edge computing
  • >> Distributed artificial intelligence with networking
  • >> Machine learning and computational intelligence techniques for                            handling big data
  • >> Information-centric networking and software-defined network for edge      intelligence
  • >> Joint radio, cache, and computing resource allocation in intelligent edge            computing
  • >> Power control, energy saving and harvesting in intelligent edge computing
  • >> Mobility estimation/enhancement/modeling in intelligent edge computing
  • >> Intelligent edge-based mobile computing and analysis
  • >> Multimedia QoS, and traffic management in intelligent networks
  • >> Applications of AI for 5G resource management
  • >> Deep-learning network approaches for wireless system applications and            services
  • >> Real-time communication interfaces and protocols
  • >> Intelligent infrastructures at the edge
  • >> Hardware testbed or field trial for AI-driven intelligent edge computing
  • >> Fog and MEC security and related considerations
